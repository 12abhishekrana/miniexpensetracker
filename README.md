💰 Mini Expense Tracker 

A modern full-stack Expense Tracker + Fintech Dashboard built using React, Node.js, Express, and JSON/MongoDB-ready backend.

It allows users to track expenses, analyze spending, and manage budget in a clean dashboard UI.

live demo: miniexpensetracker-dxud.vercel.app

🚀 Features
📊 Add / Delete / View Expenses
🗂 Category-based expense tracking
📅 Date-wise expense management
💸 Wallet & budget system
📈 Analytics dashboard (Pie / Histogram ready)
🤖 AI Finance Assistant (optional module)
🎨 Modern responsive UI (mobile + desktop)
⚡ Fast API using Express
💾 JSON / MongoDB ready backend

⚙️ Installation & Setup 

 #How to run locally
1️⃣ Clone Repo
git clone https://github.com/yourusername/miniexpense.git
cd miniexpense
2️⃣ Setup Backend
cd backend/server
npm install
npm run dev
3️⃣ Setup Frontend
cd client
npm install
npm run dev
🔐 Environment Variables

Create a .env file inside backend:

PORT=5000
MONGO_URI=your_mongodb_url_here
